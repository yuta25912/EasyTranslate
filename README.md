# EasyTranslate プラグイン

Google Translator を利用して、ボットに翻訳機能を追加します。**APIキーの設定は一切不要です。**

## ✨ 機能
- **🌍 翻訳ブロック**: 指定したテキストをドロップダウンで選択した言語に即座に翻訳します。APIキーなしですぐに使えます。

## ⚙️ 使い方
1. ツールボックスの「🌍 EasyTranslate」から「🌍 (テキスト) を (言語) に翻訳」ブロックを取り出します。
2. 翻訳したい箇所でこのブロックを使用します。

### 注意事項
- このプラグインは `deep_translator` ライブラリを使用します。
- 使用前にPCで `pip install deep_translator` を実行してください。

## 🚀 インストール方法
1. [Easy Discord Bot Builder](https://edbplugin.github.io/easy-bdp/) を開きます。
2. プラグインから以下のURLを指定してインストールしてください：
   `https://edbplugin.github.io/easy-bdp/editor/index.html?install-plugin=yuta25912%2FEasyTranslate%40master`
